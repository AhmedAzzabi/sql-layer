There are two software requirements that must be met before the FoundationDB SQL Layer can be installed.

Requirements

  • OS X version 10.7 or later
  • FoundationDB Client library version 1.0.0 or later
  • Java Runtime Environment version 1.7.0 or later

Foundation Client
This client library can be found at foundationdb.com/get#osx. Installing FoundationDB Server is also recommended.

Java Runtime Environment
The Java requirement can be satisfied by any of the Java SE Downloads. If you are unsure of which to install, choose JRE and select the dmg.
